# Founder badge assets

This directory includes proposed OpenStreetMap-NG Founder assets for the contributor badge incentive:

- `founder-badge-medallion.svg` — square/circular badge suitable for profile badge or compact placements.
- `founder-wordmark-medallion.svg` — horizontal lockup for pages that have enough room for the project name.

Design notes:

- Based on the existing OpenStreetMap-NG visual language: folded green map panels, the NG monogram, and rounded badge geometry.
- Adds a gold outer ring and `FOUNDER` ribbon so the asset is distinguishable from the main project logo at small sizes.
- Uses inline SVG only, with no remote assets, so it remains easy to preview, version, and recolor.
